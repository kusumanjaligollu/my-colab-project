# 🎓 ANITS Engineering College Chatbot 🤖

An interactive chatbot built using Python and Jupyter Notebook widgets to help users explore information about **ANITS Engineering College**.

---

## ✨ Features

* 📚 View available courses (B.Tech, M.Tech, MBA, MCA)
* 📝 Admission process details
* 💰 Fee structure overview
* 🏠 Hostel facilities information
* 📍 College location
* 💼 Placement statistics
* ❌ Exit option to end interaction

---

## 🛠️ Technologies Used

* Python 🐍
* ipywidgets (for interactive buttons)
* Jupyter Notebook / Google Colab

---

## 🚀 How to Run the Project

### Option 1: Run on Google Colab

1. Open Google Colab
2. Upload the `.ipynb` file
3. Run all cells

---

### Option 2: Run Locally

1. Install required library:

```bash
pip install ipywidgets
```

2. Open the notebook using Jupyter:

```bash
jupyter notebook
```

3. Run all cells
